Using the Linter

AVOID using braces in interpolation when not needed.

If you're just interpolating a simple identifier, and it's not immediately followed by more alphanumeric text, the {} can and should be omitted.

GOOD:

print("Hi, $name!");

BAD:

print("Hi, ${name}!");
