Using the Linter

Multiline strings are easier to read when they start with a newline (a newline starting a multiline string is ignored).

BAD:

var s1 = '''{
  "a": 1,
  "b": 2
}''';

GOOD:

var s1 = '''
{
  "a": 1,
  "b": 2
}''';

var s2 = '''This onliner multiline string is ok. It usually allows to escape both ' and " in the string.''';
